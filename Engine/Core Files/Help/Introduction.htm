Introduction

C4P stands for Categorizer for Pictures .  There are two applications that use the name “C4P” :

  • the C4P EnCase EnScript, and 
  • the C4P Windows End-User Application.

 

This document applies to the C4P EnCase EnScript.  The purpose of this document is to help EnCase users understand the options available in the EnScript.

This document does not relate to the terminology used by EnCase, or the EnScript programming language.  Any questions regarding the operation of EnCase, or EnScript in general should be directed to Guidance Software 

 

The author of C4P is Trevor Fairchild .  Mr. Fairchild is a civilian Programmer / Analyst, working in Electronic Crime Section, Ontario Provincial Police, based out of Orillia, Ontario, Canada.

 

The EnScript is provided "as is" to the EnCase community.

 

If you wish to know more about the C4P applications:  http://www.e-crime.on.ca/software/C4Pintro
